# ALGONIVE – NETFLIX / OTT USER DATA INSIGHTS  
### End-to-End AI-Powered Analytics & Recommendation Platform


## Project Overview
**ALGONIVE – Netflix / OTT User Data Insights** is a **production-grade AI analytics and recommendation platform** built using **Machine Learning, Deep Learning (Autoencoders), Semantic Search**, and an **interactive Streamlit dashboard**.

This system performs:
- Advanced **data analysis & visualization**
- **Content clustering & segmentation**
- **TF-IDF based recommendations**
- **Deep Learning based recommendation engine using autoencoders**
- **Semantic search (description → recommendations)**
- **Personalized recommender system (user taste profiling)**
- **Live search with autocomplete intelligence**
- **Enterprise-grade interactive dashboard UI**


## Key Features

### Data Analytics & Visualization
- Movie vs TV Show distribution  
- Genre trends analysis  
- Country-wise content production  
- Content growth over time  
- Interactive charts using Plotly  

### AI-Powered Recommendation Engine
- Traditional content-based recommender (TF-IDF + cosine similarity)  
- **Deep Learning recommender using autoencoders**  
- High-quality **latent embeddings for semantic similarity**  
- Hybrid recommendation pipeline  

### Semantic Search
- Users can **enter descriptions or moods**
- AI recommends **contextually relevant movies/shows**

### Personalized Recommendation System
- Builds **user taste profile**
- Generates **custom recommendations** based on preferences

### Smart UX
- Live search autocomplete  
- Professional animated loading intro  
- Premium UI with modern design system
